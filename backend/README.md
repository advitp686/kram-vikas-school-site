# Backend (Django + DRF)

## Setup

```powershell
cd backend
..\kramvikas\Scripts\python -m pip install -r requirements.txt
..\kramvikas\Scripts\python manage.py migrate
..\kramvikas\Scripts\python manage.py seed_demo_data
```

## Run

```powershell
..\kramvikas\Scripts\python manage.py runserver 127.0.0.1:8000
```

## API Base

`http://127.0.0.1:8000/api/v1`

```powershell
..\kramvikas\Scripts\python manage.py createsuperuser
```

Then open `http://127.0.0.1:8000/admin/`.

## Environment

Copy `.env.example` and configure:

```env
DJANGO_SECRET_KEY=change-me
DJANGO_DEBUG=True
DJANGO_ALLOWED_HOSTS=127.0.0.1,localhost
CORS_ALLOWED_ORIGINS=http://localhost:3000,http://127.0.0.1:3000
DJANGO_CSRF_TRUSTED_ORIGINS=http://localhost:3000,http://127.0.0.1:3000
DATABASE_URL=
DJANGO_DB_SSLMODE=require
```

If `DATABASE_URL` is empty, the project uses local SQLite. For hosted demos, set `DATABASE_URL` to PostgreSQL.

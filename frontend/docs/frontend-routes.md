# Frontend Route Map

## Public Routes

1. `/` Home
2. `/about` About Institution
3. `/method` Learning Method
4. `/classes` Class listing
5. `/classes/:slug` Class detail
6. `/worksheets` Sample worksheets
7. `/teachers` Teacher profiles
8. `/parent-corner` FAQ and schedule
9. `/admissions` Inquiry form
10. `/login` Student login
11. `/student/profile` Student personal profile (protected)
12. `/privacy`
13. `/terms`

## Shared Layout

- Sticky top navigation
- Responsive content container
- 3-column footer with contact details

## Component Requirements

- `SiteLayout`
- Cards: class, teacher, testimonial, worksheet
- Timeline block for learning method
- Inquiry form with validation and success state
- Filter controls on worksheet page
